# Prompt para rediseño del MVP AgroMap-Cereté

## Objetivo

Necesito que analices y refactorices completamente el diseño visual de este MVP, manteniendo toda su funcionalidad actual, lógica de negocio, estructura de componentes y flujo de datos.

El proyecto se llama **AgroMap-Cereté** y está enfocado en la gestión georreferenciada de cultivos periurbanos para apoyar la seguridad alimentaria estudiantil.

## Contexto del proyecto

AgroMap-Cereté nace como una solución tecnológica para monitorear cultivos de ciclo corto en predios periurbanos con el fin de fortalecer la seguridad alimentaria de estudiantes universitarios.

El sistema permite:

* Registrar predios agrícolas
* Geolocalizar terrenos con coordenadas GPS
* Monitorear cultivos de ciclo corto
* Gestionar bitácoras de seguimiento
* Visualizar reportes e indicadores
* Organizar información agrícola para mejorar la planificación de alimentos

El problema central es la falta de herramientas tecnológicas funcionales para gestionar predios periurbanos orientados a seguridad alimentaria estudiantil.
